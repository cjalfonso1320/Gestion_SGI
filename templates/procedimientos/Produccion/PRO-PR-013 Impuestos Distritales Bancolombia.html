1. OBJETIVO

Procesar integralmente los impuestos distritales recaudos por Bancolombia y dar cumplimiento al sistema de gestión integrado

2. ALCANCE

Este procedimiento aplica a los servicios de impuestos distritales implementados por Digicom System Corporation S.A

3. RESPONSABLES

Gerente de operaciones, director de procesos, jefe de Procesos, Líder de procesos, Auxiliar de procesos y Digitador

4. DEFINICIONES
  • Archivo Plano: Archivo de datos que se utiliza para intercambio de información entre las diferentes aplicaciones.
  • LOG bancario: Es el registro de las transacciones de una fecha de recaudo remitido por el banco.
  • Página Web: es un gran espacio documental organizado, dedicado al intercambio de información.
5. DOCUMENTOS DE REFERENCIA
  • Acuerdo de servicios.
  • Norma NTC ISO 9001.
  • Resolución SHD 167 26 de octubre de 2018
