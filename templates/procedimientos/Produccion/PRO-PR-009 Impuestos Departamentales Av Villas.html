1. OBJETIVO

Procesar integralmente los formularios de Impuestos Departamentales recaudados y pactados con la entidad Bancaria y dar cumplimiento al Sistema de Gestión

2. ALCANCE

Este procedimiento aplica a los servicios de Impuestos Departamentales Av. Villas (Cundinamarca, Nariño y Valle), implementados por Digicom System Corporation S.A

3. RESPONSABLES

Gerente de operaciones, Director de proceso, Líder de proceso, Auxiliar de procesos y Digitador

4. DEFINICIONES

 FTP: es un protocolo de transferencia de archivos entre sistemas conectados a una red TCP (protocolo de control de transmisión) basado en la arquitectura cliente-servidor, de manera que desde un equipo cliente nos podemos conectar a un servidor para descargar archivos desde el o para enviarle nuestros propios archivos independientemente del sistema operativo utilizado en cada equipo.  Archivo Plano: archivo de datos que se utiliza para el intercambio de información entre las diferentes aplicaciones.  Conciliación: registro consolidado de transacciones por convenios.  Log Bancario: es el registro de las transacciones de una fecha de recaudo.  Precinto: Zuncho de seguridad numerado que una vez colocado en el medio de transporte de documentos o valores no puede ser violentado ni roto excepto por el destinatario.  Tula: empaque en material de lona que permite el transporte seguro de documentos
