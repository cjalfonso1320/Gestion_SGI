1. OBJETIVO

Procesar, Validar y administrar la información de los Convenios recaudados por el Banco Davivienda, en cumplimiento con los requerimientos del Sistema de Gestión Integrado

2. ALCANCE

Este procedimiento aplica a los Convenios (Colfondos, Movilidad Barranquilla, Fontic, Municipales Cali, Municipales Soacha ReteIca, Sobretasa a la Gasolina y Municipales Cartagena) entregados por el Banco Davivienda, para el procesamiento de la información

3. RESPONSABLES

Gerente de Operaciones, Director de Procesos, Líder de procesos, Auxilia de procesos y Digitador

4. DEFINICIONES

• Archivo Plano: Archivo de datos que se utiliza para intercambio de información entre las diferentes aplicaciones. • LOG bancario: Es el registro de las transacciones de una fecha de recaudo. • Página Web: es un gran espacio documental organizado dedicado al intercambio de información. • Enzipar: Es la carpeta comprimida la cual contiene las plantillas de expedientes y garantías

5. DOCUMENTOS DE REFERENCIA

➢ Acuerdo de servicios. ➢ Norma NTC-ISO-9001
