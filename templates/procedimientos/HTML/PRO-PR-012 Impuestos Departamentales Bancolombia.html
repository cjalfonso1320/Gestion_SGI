IMPUESTOS DEPARTAMENTALES BANCOLOMBIA

1. OBJETIVO

Procesar integralmente los formularios de Impuestos Departamentales recaudados por Bancolombia y dar cumplimiento al Sistema de Gestión Integrado

2. ALCANCE

Este procedimiento aplica a los servicios de Impuestos Departamentales recaudados por Bancolombia e implementados por DIGICOM System Corporation S.A

3. RESPONSABLES

Gerente de Operaciones, jefe de proceso y Líder de Proceso, Auxiliar de Procesos, Digitador

4. DEFINICIONES

Archivo Plano: Archivo de datos que se utiliza para intercambio de información entre los diferentes aplicativos. LOG bancario: Es el registro de las transacciones de una fecha de recaudo Precinto: Suncho de seguridad numerado que una vez colocado en el medio de transporte de documentos o valores no puede ser violentado ni roto excepto por el destinatario
