11/08/2023 CONVENIOS ESPECIALES DAVIVIENDA

1. OBJETIVO

Procesar, Validar y administrar la información de los Convenios Especiales recaudados por el Banco Davivienda, cumplimiento con los requerimientos implementados por el Sistema de Gestión

2. ALCANCE

Este procedimiento aplica a los Convenios Especiales entregados por el Banco Davivienda, para el procesamiento de la información

3. RESPONSABLES

Gerente de Operaciones, Director de Procesos, Líder de procesos, Auxiliar de procesos y Diigitador

4. DEFINICIONES

• Archivo Plano: Archivo de datos que se utiliza para intercambio de información entre las diferentes aplicaciones. • LOG bancario: Es el registro de las transacciones de una fecha de recaudo. • Página Web: es un gran espacio documental organizado dedicado al intercambio de información. • Precinto: Suncho de seguridad numerado que una vez colocado en el medio de transporte de documentos o valores no puede ser violentado ni roto excepto por el destinatario. • Enzipar: Es la carpeta comprimida la cual contiene las plantillas de expedientes y garantías

5. DOCUMENTOS DE REFERENCIA

➢ Acuerdo de servicios. ➢ Norma NTC-ISO-9001
